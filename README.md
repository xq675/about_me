# About Me: Sia Qiu

This project is a personal, multi-page website designed to introduce Sia's background, interests, and creative pursuits. The site features three main sections using three different pages, including the Introduction page, the Music page, and the Travel page.

## Live Demo
https://xq675.github.io/about_me/

## Technologies
- HTML
- CSS

## Key Features
This project demonstrates fundamental front-end development using a single CSS stylesheet.
- Responsive Images: “srcset” attribute is used to provide responsive raster graphics to optimize load time and display quality
- Web Font and Links: a self-hosted font is applied using CSS, and navigational links are customized with various link states
